Bank of America App Redesign

UI / Visual Design

Overview

This project was a an exploration of visual design. I was to define and develop unique visual design concepts through the exploration and expression of a mobile application. We were to use an existing product, as to not be completely starting from scratch, and mainly focusing on the visual design instead of adding/removing features or modifying workflows.

Problem

The original app is functional, but by no means appealing. I, as well as other users, mainly use it to look up account balances and recent transactions, important information that is not displayed well with the current design. There is very grouping and use of white space between elements and key hierarchy is missing overall.

Solution

Banking doesn't have to be so rigid, I wanted my designs to feel fresh and approachable, leaving users calm while utilizing the app instead of running away from a flat, boxy, type heavy interface as soon as possible. I sought to use dimension and feel in my visual design as well as establish hierarchy among elements
